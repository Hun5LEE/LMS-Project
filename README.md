# 1. 프로그램 소개
> 학습자의 학습진행도를 파악하는 웹 플랫폼

멘토의 피드백과 수강생의 질문/커뮤니티 활동이 자유롭도록 기존의 다른 LMS 시스템에서
과제/피드백 기능을 탑재하여 제작한 스나이퍼팩토리 전용 커스텀 LMS`Learning Managment System`입니다.
이러한 IT 서비스 기반의 학습관리 시스템을 통해 멘토와 수강생 간의 양방향 소통을 추구하고자 합니다.
